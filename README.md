# JoJo's Bizarre Adventure - 角色与替身能力查询

一个关于《JoJo的奇妙冒险》的 Python 项目，包含系列主要角色信息和替身能力数据。

## 功能特性

- 查询 JoJo 系列各部主角信息
- 查询角色的替身（Stand）能力
- 按替身能力关键字搜索角色
- 展示各部剧情简介

## 快速开始

```bash
python main.py
```

## 项目结构

```
jojo/
├── main.py          # 主程序入口
├── characters.py    # 角色数据与查询逻辑
└── README.md        # 项目说明
```

## 数据来源

角色信息整理自《JoJo的奇妙冒险》漫画及动画系列（荒木飞吕彦著）。
